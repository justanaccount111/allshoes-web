Proyecto Web AllShoes
Descripción

Este proyecto consiste en el desarrollo de un sitio web estático utilizando HTML y CSS, basado en un mockup proporcionado en la actividad.

La web representa la página de una marca ficticia de calzado llamada AllShoes, e incluye diferentes secciones informativas, un catálogo de productos, cursos relacionados con el sector y una página de contacto con formulario.

El objetivo del proyecto es aplicar los conocimientos básicos de estructura HTML y maquetación con CSS.

Tecnologías utilizadas

HTML5

CSS3

No se han utilizado frameworks ni librerías externas.

Estructura del proyecto

index.html → página principal

sobre-nosotros.html → información sobre la empresa

cursos.html → presentación de cursos

tienda.html → catálogo de productos

contacto.html → información de contacto y formulario

Carpetas principales:

css/ → hoja de estilos (estilos.css)

img/ → imágenes utilizadas en el sitio

Características

Maquetación mediante Flexbox

Diseño basado en el mockup proporcionado

Cabecera y pie de página comunes en todas las páginas

Página de contacto con formulario

Diseño adaptable a diferentes tamaños de pantalla

Autor

Proyecto desarrollado como actividad práctica de HTML y CSS.
